# WISCO POI Tracking System
## วิธีติดตั้งและใช้งาน (Setup Guide)

---

## 📁 โครงสร้างไฟล์ (File Structure)

```
wisco-server/
├── start.bat          ← ดับเบิ้ลคลิกเพื่อเปิดเซิร์ฟเวอร์ (Windows)
├── server.js          ← ไฟล์ backend หลัก
├── package.json       ← รายการ packages ที่ต้องใช้
├── wisco_poi.db       ← ฐานข้อมูล (สร้างอัตโนมัติตอนรันครั้งแรก)
└── public/
    ├── dashboard.html ← หน้าจอสำหรับห้องผู้จัดการ
    └── edit.html      ← หน้าสำหรับพนักงานกรอกข้อมูล
```

---

## 🖥️ ขั้นตอนการติดตั้ง (Installation Steps)

### ขั้นตอนที่ 1 — ติดตั้ง Node.js
1. ไปที่ **https://nodejs.org**
2. ดาวน์โหลด **LTS version** (ปุ่มซ้าย)
3. ติดตั้งด้วยค่า default ทั้งหมด (กด Next ไปเรื่อยๆ)
4. Restart คอมพิวเตอร์

### ขั้นตอนที่ 2 — วางไฟล์ลงคอมพิวเตอร์โฮสต์
1. คัดลอกโฟลเดอร์ `wisco-server` ทั้งหมดไปไว้ที่ `C:\wisco-server\`

### ขั้นตอนที่ 3 — เปิดเซิร์ฟเวอร์
1. ดับเบิ้ลคลิกที่ไฟล์ **`start.bat`**
2. ครั้งแรกจะติดตั้ง packages อัตโนมัติ (ใช้เวลา 1-2 นาที)
3. เมื่อเห็นข้อความ **"✅ WISCO POI Server is running!"** แสดงว่าพร้อมใช้งาน

### ขั้นตอนที่ 4 — หา IP Address ของคอมพิวเตอร์โฮสต์
1. กด `Windows + R` พิมพ์ `cmd` แล้วกด Enter
2. พิมพ์ `ipconfig` แล้วกด Enter
3. มองหา **IPv4 Address** เช่น `192.168.1.105`

### ขั้นตอนที่ 5 — เปิดใช้งานจากคอมพิวเตอร์อื่นในออฟฟิศ

| หน้า | URL |
|------|-----|
| 📊 Dashboard (ห้องผู้จัดการ) | `http://192.168.1.105:3000/dashboard.html` |
| ✏️ Edit Page (พนักงาน) | `http://192.168.1.105:3000/edit.html` |

> ⚠️ เปลี่ยน `192.168.1.105` เป็น IP จริงของคอมพิวเตอร์โฮสต์

## 🔄 การเปิดเซิร์ฟเวอร์ทุกวัน
- ดับเบิ้ลคลิก **`start.bat`** ทุกครั้งที่เปิดคอมพิวเตอร์
- อย่าปิดหน้าต่าง Command Prompt (กระตุก = เซิร์ฟเวอร์หยุด)
- ถ้าต้องการให้เปิดอัตโนมัติตอน Windows เริ่ม ให้เพิ่ม shortcut ของ `start.bat` ไปที่:
  `C:\Users\[ชื่อUser]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`

## 🛠️ แก้ปัญหาเบื้องต้น (Troubleshooting)

| ปัญหา | วิธีแก้ |
|-------|---------|
| เปิด start.bat แล้วปิดไปเอง | คลิกขวา → Run as administrator |
| "node is not installed" | ติดตั้ง Node.js จาก nodejs.org |
| เปิดจากคอมอื่นไม่ได้ | ตรวจสอบว่าอยู่ใน Wi-Fi เดียวกัน และ IP ถูกต้อง |
| Port 3000 ถูกใช้แล้ว | เปลี่ยน PORT=3000 ใน server.js เป็น 3001 หรืออื่นๆ |
| ข้อมูลหาย | ไฟล์ wisco_poi.db คือฐานข้อมูล อย่าลบหรือย้าย |

- ไฟล์นี้คือข้อมูลทั้งหมดของระบบ
